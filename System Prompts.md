System prompts are prompts given to the LLMs before it starts interacting with user. Its like global context of LLM. It can be security rules, safety fallbacks to improve accuracy and it mitigates risk when it eventually interact with user inputs.

It can be used to give LLM a character to follow for interacting with user. It can be used to bound LLM into rules. I can be used to give context and domain knowledge. We can also mention rubrics or framework LLM can use to asses and evaluate.

In short system prompts are set of instructions that LLM will follow in every conversation with user.
